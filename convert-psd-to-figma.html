How to convert PSD to Figma

Use PSD2Figma when you need to move an Adobe Photoshop layout into Figma as editable layers instead of manually rebuilding every screen.

Install free pluginGet unlimited conversions

Conversion workflow

  1. Install PSD ↔ Figma — Free Converter from Figma Community.
  2. Open any Figma design file.
  3. Run the plugin from the Plugins menu.
  4. Select Import PSD.
  5. Drop your `.psd` file or browse from disk.
  6. Wait for PSD2Figma to create editable Figma objects.
  7. Review text, images, masks, effects and component candidates.

What to check after import

Fonts

If a Photoshop font is not installed locally, Figma may substitute it. Review typography before handoff.

Effects

PSD effects are mapped to Figma approximations. Complex Photoshop-only effects may need small manual cleanup.

Masks

Raster and vector masks are imported where possible. Inspect clipping behavior on complex layouts.

Components

Repeated objects can become components, but you should still rename and organize them for your design system.
