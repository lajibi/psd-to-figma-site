PSD import

Import PSD files into Figma

Figma does not natively open PSD files. PSD2Figma adds a plugin-based import path so you can bring Photoshop layouts into Figma as editable design objects.

Import PSD without uploading client files

The plugin parses the PSD in the browser context and creates Figma nodes through the Figma Plugin API. Your design file is not uploaded to PSD2Figma servers.
