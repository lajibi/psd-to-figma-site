What it does

A Figma plugin that converts between Adobe Photoshop (.psd) files and Figma designs. Import PSD files into Figma with full layer fidelity, or export Figma frames back to layered PSD files.

All processing happens locally in your browser. Your files are never uploaded to any server.

Features

PSD to Figma (Import)

  • Full layer structure — groups, images, text, shapes with correct hierarchy
  • Component detection — identifies repeated elements, converts to Figma Components
  • Mixed-style text — per-character font, size, color, letter-spacing
  • Effects — shadows, glow, stroke, bevel/emboss, satin, color/gradient overlay
  • Gradient fills — linear, radial, angular, diamond
  • Layer masks — raster and vector masks
  • Smart objects — embedded content extracted as images
  • Smart font mapping — Adobe CJK fonts mapped to system equivalents

Figma to PSD (Export)

  • Layer structure preserved — frames, groups, text, shapes as PSD layers
  • Text with full styling — font, size, color, tracking, leading, decorations
  • Effects — shadows, strokes converted to PSD layer effects
  • Gradients — linear, radial, angular with color stops
  • Blend modes — all 19 standard modes mapped

How to use

Import PSD

  1. Open any Figma design file
  2. Run the plugin (Plugins → PSD ↔ Figma)
  3. Select "Import PSD" mode
  4. Drop your .psd file or click to browse
  5. Done — your PSD layers are now editable Figma objects

Export PSD

  1. Select a frame in Figma
  2. Run the plugin, select "Export PSD" mode
  3. Click "Export to PSD"
  4. PSD file downloads automatically
